# Vue 3 + Vite

本项目基于 Vite 构建，使用 Vue 3 作为前端框架，采用 `<script setup>` 语法风格。

## 启动开发服务器

```bash
npm run dev
```

默认开发端口：**3001**

如需更改端口，可在 `vite.config.js` 中修改 `server.port` 配置。
